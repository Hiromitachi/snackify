Vercel offers the easiest deployment with automatic updates from GitHub.

### Step 1: Sign Up / Login
1. Go to: https://vercel.com/signup
2. Click "Continue with GitHub"
3. Authorize Vercel to access your GitHub account

### Step 2: Import Project
1. Click "Add New..." â†’ "Project"
2. Find and select **snackify** from your repositories
3. Click "Import"

### Step 3: Configure & Deploy
1. **Framework Preset**: Vite (should auto-detect)
2. **Root Directory**: `./` (leave as is)
3. **Build Command**: `npm run build` (auto-filled)
4. **Output Directory**: `dist` (auto-filled)
5. Click **"Deploy"**

### Step 4: Wait for Deployment
- Vercel will build and deploy your app (takes 1-2 minutes)
- You'll get a live URL like: `snackify-xxx.vercel.app`

### Step 5: Custom Domain (Optional)
- In project settings, you can add a custom domain
- Or use the free Vercel subdomain
